# string-program-cpp
Operating systems task to reverse, sort, find length, add 2 in the ascii of each character and Capitalize string using series of processes

<br />
It creates series of processing using fork() and then it uses exec() to call different codes that perform following functionalities <br />
1- Reverse <br />
2- Find Length <br />
3- Add 2 in ASCII of each character
4- Sort
5- Capitalize
